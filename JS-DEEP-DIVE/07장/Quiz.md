### 🙋🏻‍♀️ Quiz

```jsx
console.log(0 == "0"); // true
console.log(0 === "0"); // false
```

느슨한 비교(`==`)와 엄격한 비교(`===`)의 차이✨

```jsx
let a = 5;
let b = a++;
let c = ++a;

console.log(a, b, c);

// 5, 7, 7
```

- **`a++`** (후위 증가 연산자):
  - `b = a++`에서 `b`는 증가하기 전의 `a` 값을 받는다.
  - 이후, `a`가 1 증가한다.결과적으로, `b = 5`, `a = 6`이 된다.
- **`++a`** (전위 증가 연산자):
  - `c = ++a`에서 `a`는 먼저 1 증가한 뒤에 값이 할당된다.
  - `a`는 이미 6이었으므로 1 증가하여 `7`이 되고, `c = 7`이 된다.
- **최종 상태**:
  - `a = 7`, `b = 5`, `c = 7`
