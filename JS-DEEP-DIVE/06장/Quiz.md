### 🙋🏻‍♀️ Quiz

```jsx
console.log(1 + "2");
console.log("5" - 2);
console.log(true + false);

// "12", 3, 1
```

1. **`1 + "2"`**:
   - 숫자 `1`과 문자열 `"2"`가 만나면 숫자가 문자열로 변환된다.
   - 결과: `"12"`
2. **`"5" - 2`**:
   - 문자열 `"5"`가 숫자로 변환되어 계산된다.
   - 결과: `3`
3. **`true + false`**:
   - `true`는 숫자 `1`로, `false`는 숫자 `0`으로 변환된다.
   - 결과: `1 + 0 = 1`
